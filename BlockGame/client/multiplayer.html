How to Play

Place the blocks so that a row is filled. When filled you are awarded one point and that row is deleted.
After every 25 seconds it will alter the game a bit by either, flipping the board, leaning the board left or right, jumbling the rows, mirroring the board or selecting a new piece.
Use a s d keys to shift the block that is dropping one block over.
Use e key to rotate the block that is dropping.
Connect to the server by pressing the connect buttont at the button of the tetris canvas.
When the game ends press the restart button to restart the game.
